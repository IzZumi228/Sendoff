# Sendoff
Hackbeta 2026 Project: Hero Search Engine
* Group: DEA

Given certain situations search through a database (from a .csv file) to find for the best heros to sucessfully complete the mission.

## How to Use
### Backend
This project uses Python version `3.12`

Dependencies:
```
pip install fastapi pydantic pandas numpy uvicorn
```

Running the server
```bash
cd backend/
uvicorn main:api --reload
```
The .csv file is saved in `backend/src/` as `hackbeta-herodata.csv`
